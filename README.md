# 🥔 Potato Chat Bot (aiogram 3.x)

Интерактивный игровой бот для Telegram-групп с элементами экономики, азартных игр и системой наказаний.

## 🚀 Основные возможности
- **Экономика:** Копание картошки (`/dig`), передача валюты (`/give`).
- **Игры:** Блэкджек (`/bj`), Классическая рулетка (`/roulette`), КНБ между игроками (`/rps`), Дуэли (`/pvp`) и ежедневная игра `/pidor`.
- **Магазин:** Покупка щитов от мута, лицензий на титулы и страховок для казино.
- **Инвентарь:** Система кастомных префиксов в топе (`/settitle`).
- **Админка:** Управление ценами, КД и выдачей картошки (`/add`).
- **Антиспам:** Встроенная защита чата от флуда.

## 🛠 Технологии
- **Язык:** Python 3.12
- **Фреймворк:** [aiogram 3.x](https://github.com/aiogram/aiogram)
- **База данных:** SQLite3
- **Контейнеризация:** Docker & Docker Compose
- **CI/CD:** GitHub Actions (автоматический деплой на сервер)

## 📦 Быстрый запуск (Local)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ВашЛогин/potato_chat_bot.git
   cd potato_chat_bot
   ```
2. Создайте файл `.env` и заполните его:
   ```env
   BOT_TOKEN=ваш_токен_от_BotFather
   MY_ID=ваш_телеграм_id
   PROXY_URL=http://... (если нужно)
   ```
3. Запустите через Docker:
   ```bash
   docker compose up -d --build
   ```

## 🤖 Команды
- `/start` — Команды бота
- `/dig` — Найти картошку (зависит от КД в настройках).
- `/top` — Таблица лидеров чата с учетом винрейта.
- `/zaim` — Взять 50 🥔 в долг на 30 минут (иначе мут).
- `/shop` — Открыть лавку предметов.
- `/sleep` — Отправить игрока «спать» (мут) за картошку.
- `/roulette [ставка] [тип]` — Ставки на числа, цвета, четность и дюжины.
- `/all` — Пингануть всех
- `/pvp` — Вызвать кого нибуть на дуэль(сразу или с подтверждением).
- `/rps` — камень ножницы бумага на картошку
- `/bj` — блекджек
- `/flip` — 50/50 - coinflip
- `/give` — дать кому нибуть картошку
- `/pidor` — pidor дня

## ⚙️ CI/CD (GitHub Actions)
В проекте настроен автоматический деплой. При каждом `git push` в ветку `main`:
1. GitHub подключается к серверу по SSH.
2. Подтягивает изменения через `git pull`.
3. Пересобирает и перезапускает Docker-контейнеры.

Для работы требуются следующие **Secrets** в GitHub:
`HOST`, `USERNAME`, `SSH_KEY`, `PORT`, `WORK_DIR`.
